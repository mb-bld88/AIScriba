
# 🔑 How to get your Google Gemini API Key

AIScriba uses Google Gemini Artificial Intelligence to transcribe and summarize meetings. To function, each user (or the company admin) must provide a valid API Key.

Google currently offers a generous **free tier** via Google AI Studio.

### Step-by-Step Procedure

1.  Go to **Google AI Studio**: [https://aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)
2.  Log in with your Google Account (Personal or Workspace).
3.  Click the blue button **"Create API Key"**.
4.  Select an existing Google Cloud project or click **"Create API key in new project"**.
5.  Copy the string starting with `AIza...`. This is your **API Key**.

### Configure in AIScriba

1.  Log in to AIScriba.
2.  Click the **Gear Icon (Settings)** in the top right corner.
3.  In the **General** tab, find the field "Your Google Gemini API Key".
4.  Paste the key you just copied.
5.  Click **Save**.

Done! You can now record and generate meeting minutes.
